# PhonoPlay
 browser-based music player that allows users to upload and play local audio files with playlist support and advanced playback controls.

## Features

- **Upload Songs** - Upload audio files directly from your device
- **Playlist Management** - View and manage your uploaded songs
-  **Playback Controls** - Play, pause, next, previous
- **Shuffle Mode** - Random playback order
- **Loop Mode** - Repeat songs or entire playlist
- **Progress Tracking** - Visual progress bar with time display
- **Session Memory** - Remembers last played song and position using localStorage
- **Minimalist Design** - Clean black and grey theme with parallax effects
